# 记忆小游戏

哈工大 大连实训内容

Java EE编写

awt 制作界面

分三个难度

每次选出对应数量相同图标即可通关

中途中断之后重新开始

有提示功能和计时功能

jar文件可以直接运行

源码：[https://github.com/Xbean1028/Memory-game](https://github.com/Xbean1028/Memory-game)
